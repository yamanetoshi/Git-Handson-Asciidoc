# Git-Handson-Asciidoc

## How to build

```
$ bundle install --path vendor/bundle
$ bundle exec asciidoc git-handson.adoc
$ bundle exec asciidoc git-handson2.adoc
$ bundle exec asciidoc git-handson3.adoc
```
